# Gyaanly – Online Course Selling Platform

Gyaanly is a full-stack web-based e-learning platform developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application enables instructors to upload and manage courses while allowing students to browse, purchase, and access educational content through a secure and responsive interface.

The platform is designed to bridge the gap between educators and learners by providing an accessible and scalable digital marketplace for online education.

---

## Overview

With the growing demand for flexible learning solutions, Gyaanly provides a centralized platform where:

- Instructors can create and manage online courses
- Students can explore and purchase courses
- Users can securely access purchased learning material
- Administrators can manage platform content

The system follows a modular client-server architecture and implements secure authentication mechanisms to ensure safe access and data protection.

---

## Key Features

- User Registration and Authentication
- Role-Based Access Control (Admin / Student)
- Course Listing and Management
- Secure Login and Signup using JWT
- Password Encryption using bcrypt
- Course Purchase Functionality
- Student Dashboard for Purchased Courses
- Responsive User Interface
- RESTful API Integration

## System Architecture

Gyaanly follows a three-tier architecture:

- Presentation Layer – React.js based frontend
- Application Layer – Node.js & Express.js REST APIs
- Data Layer – MongoDB Database

The frontend communicates with the backend through RESTful APIs for real-time data exchange and secure operations.
